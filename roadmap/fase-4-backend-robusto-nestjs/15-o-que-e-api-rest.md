# 15 - Bem-vindo à API REST 🌐

O seu cronômetro salva no banco, mas como um navegador (onde o usuário clica) vai falar com o seu código de banco de dados?
Eles precisam conversar através da internet via **API REST**.

A internet funciona basicamente com Verbos HTTP:
*   `GET`: Me dá uma informação (Ex: pegar o histórico de pomodoros).
*   `POST`: Toma essa informação, salva aí! (Ex: salvar que acabei um pomodoro de 25 min).

Para criar essas "portas" (rotas), usaremos o framework **NestJS**. Ele é o queridinho das grandes empresas porque obriga você a deixar o código super organizado.

Siga para o próximo passo para ligar o servidor Nest!
