# Pomodoro TS - Do Zero à Produção 🍅

Bem-vindo ao **Pomodoro TS**, um projeto educacional open-source projetado para ensinar desenvolvimento Full-Stack moderno do básico à infraestrutura.

O objetivo deste projeto é guiar desenvolvedores (mesmo com conhecimento básico de programação) através da construção de um sistema completo de Pomodoro, ensinando não apenas a codificar, mas a entender arquitetura de software corporativa.

---

## 🛠️ A Nossa Stack Tecnológica
- **Linguagem**: TypeScript (Forte tipagem e OOP)
- **Frontend**: Next.js (App Router), React, Tailwind CSS
- **Backend**: NestJS, Zod (Validação), Pino (Logs Estruturados)
- **Banco de Dados**: PostgreSQL com Prisma ORM
- **Infraestrutura**: Docker e Docker Compose

---

## 🗺️ Como Funciona a Trilha de Aprendizado?

O repositório está dividido em duas partes fundamentais:

### 1. O Roadmap (`/roadmap`)
Uma trilha fracionada em 7 fases. Você deve consumir o conteúdo na ordem, executando cada passo na sua pasta pessoal.
*   **Fase 1**: O Alicerce (TypeScript & Lógica)
*   **Fase 2**: Pensando em Objetos e Tempo (Core do Pomodoro)
*   **Fase 3**: A Fundação dos Dados (Docker & Prisma ORM)
*   **Fase 4**: O Backend Robusto (NestJS + Zod + Pino)
*   **Fase 5**: O Rosto do Projeto (Next.js & React)
*   **Fase 6**: O Casamento (Integração Final Front e Back)
*   **Fase 7**: Containerização para Produção (Dockerificando o seu App)

### 2. O Template Base (`/_template`)
Esta é a fundação do seu projeto. Ela já contém o "esqueleto" do frontend e do backend.

**Como começar sua jornada:**
1. Copie a pasta inteira `_template` para a raiz deste repositório.
2. Renomeie a pasta para o seu nome (exemplo: `participantes/lucas` ou apenas `lucas`).
3. Entre na sua nova pasta, leia o `README.md` que está lá dentro e siga as instruções para iniciar os servidores locais.
4. Abra os arquivos na pasta `roadmap/` e comece os desafios!

---
> *Dica: O aprendizado real vem da prática. Não apenas leia os guias do roadmap, digite e execute o código na sua própria pasta! Boa jornada!*
